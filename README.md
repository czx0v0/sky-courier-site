# sky-courier-site
## 项目运行说明
1. python == 3.9.21 
2. 克隆仓库
```bash
git clone https://github.com/czx0v0/sky-courier-site.git
```
3. 进入目录
```bash
cd sky-courier-site
```
4. 需安装库见 requirement.txt(可选手动安装) 
```bash
# 直接运行可能会出错，可以手动安装
pip install -r requirements.txt
```
5. 🌟在根目录下，运行
```bash
streamlit run sky-courier-site.py
```
